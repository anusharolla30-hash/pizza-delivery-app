# 🍕 Pizza Delivery App

## 🚀 Features
- User Registration & Login
- Add to Cart
- Place Order
- Admin Order Panel

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## ▶️ How to Run

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm start

## 📌 Notes
- MongoDB should be running locally
- Default port: 5000
